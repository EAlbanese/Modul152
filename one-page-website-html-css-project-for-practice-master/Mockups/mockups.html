Wireframes und Mockups

Wireframes und Mockups sind grundsätzlich verschieden. Ein Wireframe ist nur ein sehr grober Entwurf der zukünftigen Internetpräsenz, wohingegen das Mockup eine detaillierte Ausarbeitung darstellt.

Wireframes

Ein Wireframe ist eine sehr vereinfachte Repräsentation des Designs einer Website. Er wird verwendet um die Positionierung und Größe der Elemente einer Internetseite darzustellen. Das Ziel ist es, die einzelnen Elemente innerhalb des Layouts anzuordnen.

Mockups

Mockups werden auf der Grundlage eines Wireframes erstellt und sind viel detaillierter. Du kannst dir ein Mockup wie einen Screenshot deiner späteren Website vorstellen. In ihm werden bereits die verwendete Typografie und das Farbschema abgebildet.
